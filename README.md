## 项目简介

这是苏州中学树莓社为其虚拟看板娘「树莓酱」打造的官方展示网站。网站采用现代化的单页设计，以二次元风格展现树莓酱的魅力。

<div align="center">
  <img src="https://szzxshumei.com/wp-content/uploads/2026/04/0000102a-b191-7f6a-40ea-7d658244ce66.jpg" width="300" style="border-radius: 16px; margin: 20px;">
</div>

## 主要功能

### 🎬 启动动画
- 精美的加载界面
- Logo 循环切换动画
- 进度条加载效果
- 平滑过渡到主内容

### 🖼️ 立绘展示
- 多款树莓酱立绘可切换
- 包含 2026 新款、2025 经典、冬季限定、2021 怀旧等版本
- 标注画师信息

<div align="center">
  <img src="https://szzxshumei.com/wp-content/uploads/2026/04/00001029-5344-7bba-44c0-ba78c121c790.jpg" width="180" style="border-radius: 12px; margin: 8px;">
  <img src="https://szzxshumei.com/wp-content/uploads/2026/04/00001029-5346-75e4-4f05-4d2a4ddbb739.jpg" width="180" style="border-radius: 12px; margin: 8px;">
</div>

### 📋 角色档案
- 基本信息 (生日、年龄、个人账号)
- 行为观察
- 核心语录

### 🎨 二创作品画廊
- 展示粉丝投稿的二创作品
- 支持横向滚动浏览
- 作品包含插画、海报、红包封面、表情包、GIF 动画等

  <img src="https://szzxshumei.com/wp-content/uploads/2026/04/00001029-8ba4-721a-4168-f5f6001379c1.jpeg" width="180" style="border-radius: 12px; margin: 8px;">
  <img src="https://szzxshumei.com/wp-content/uploads/2026/04/0000102a-88d4-7984-4d6d-63e98b485720.jpg" width="180" style="border-radius: 12px; margin: 8px;">

</div>

### ✉️ 投稿页面
- 鼓励用户投稿二创作品
- 风格不限 (插画、同人文、漫画、表情包等)

### 📱 响应式设计
- 完美适配桌面端和移动端
- 侧边导航在移动端变为底部导航
- 滚动捕捉效果

### 📊 访问统计
- 集成不蒜子统计
- 显示总访问量和独立访客数

## 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架
- **原生 JavaScript** - 交互逻辑
- **Noto Serif SC & ZCOOL KuaiLe** - 字体
- **不蒜子** - 访问统计

## 部署

项目已配置 GitHub Pages 自动部署，位于 `.github/workflows/static.yml`。
